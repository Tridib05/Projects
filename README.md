# Scientific Calculator Research Project

This project is a polished research-oriented calculator application built in Python using `customtkinter`.

## Key Features

- Standard calculator UI with scientific functions
- History and memory support
- Programmer mode with BIN/OCT/HEX/DEC conversions
- Graph plotter for functions of `x`
- Matrix calculator for addition and multiplication
- Equation solver for algebraic expressions
- Theme and settings support

## Project Structure

- `main.py` - application entry point
- `ui.py` - user interface and tabbed workspace
- `calculator.py` - calculation engine, history, memory, and math utilities
- `requirements.txt` - Python dependencies
- `paper_outline.md` - research paper structure and phases

## Setup

1. Create a virtual environment:
   ```powershell
   python -m venv .venv
   .\.venv\Scripts\Activate.ps1
   ```
2. Install dependencies:
   ```powershell
   pip install -r requirements.txt
   ```
3. Run the application:
   ```powershell
   python main.py
   ```
